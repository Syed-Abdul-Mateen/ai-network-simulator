# Project Report: Cyber Attack Path Analyzer

## Overview
The Cyber Attack Path Analyzer is a security tool that simulates attacks, analyzes logs, builds attack graphs, and suggests remediation.

## Methodology
1. **Attack Simulation**: The engine performs scanning, enumeration, credential attacks, privilege escalation, and lateral movement.
2. **Log Analysis**: Logs are parsed, features extracted, and root causes identified using machine learning.
3. **Visualization**: Attack paths are rendered as graphs and reports.
4. **Response**: Remediation scripts and recommendations are generated.

## Results
- Successfully identified common attack paths.
- Provided actionable remediation steps.
- MITRE ATT&CK mapping included.
